# 🚀 ETL Pipeline Cryptocurrency Data

This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline using Python and PostgreSQL.

## 📌 Overview
The pipeline extracts cryptocurrency market data from CoinGecko API, transforms it using Pandas, and loads it into a PostgreSQL database.

## ⚙️ Tech Stack
- Python
- Pandas
- Requests
- SQLAlchemy
- PostgreSQL
- Docker

---

## 🔄 ETL Pipeline Flow

1. Extract data from CoinGecko API
2. Transform data (cleaning & structuring)
3. Load data into PostgreSQL
